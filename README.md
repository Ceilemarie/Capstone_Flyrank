# Capstone Project Workspace - FlyRank Direct Concept

This repository serves as the foundational development workspace for my undergraduate Capstone Project. As a graduating student and Frontend AI Engineer Intern, this project focuses on building a high-transparency, anti-ghosting job board designed to eliminate generic job descriptions and automated AI rejection systems.

##  Core System Concept & Problem Statement
Modern online job hunting is plagued by vague, templated job requirements ("must be a rockstar, etc.") and automated AI parsers that auto-reject applicants before a human ever sees their work. This system flips the script by enforcing absolute data clarity and human-driven evaluation:

* **Strict Requirement Enforcement:** A job posting interface that rejects generic templates. Companies must provide exact headcounts (e.g., "Hiring exactly 3 people") and rigid technical criteria with zero filler text.
* **Verified Human Evaluation (Anti-AI Gatekeeping):** The platform bypasses automated AI screening. It features a transparent tracking pipeline showing the applicant exactly when an *active, verified human recruiter* opens, reviews, and updates their application status.
* **Frontend Focus:** Crafting a highly responsive, data-driven dashboard that gives fresh graduates complete clarity over their application status, removing the anxiety of the traditional online job hunt.
